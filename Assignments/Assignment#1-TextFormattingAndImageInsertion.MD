## Task Overview

You are required to create an article using various HTML tags to demonstrate text formatting. Please follow the guidelines below:

### Article Structure

- **Main Heading:** Use the `<h1>` tag for the main heading of your article.
- **Sub-Headings:** Create five sub-headings, each with a specific `<h>` tag, from `<h2>` to `<h6>`. Ensure that each sub-heading is appropriately nested and formatted.

### Paragraphs and Formatting

- **Sub-Heading 1:** Use the `<h2>` tag.  
  Provide one paragraph of text under this sub-heading.  
- **Sub-Heading 2:** Use the `<h3>` tag.  
  Include both italic `<i>` and bold `<strong>` text within the paragraph under this sub-heading.
- **Sub-Heading 3:** Use the `<h4>` tag.  
  Provide one paragraph of text under this sub-heading.  
- **Sub-Heading 4:** Use the `<h5>` tag.  
  Apply italic `<em>` and bold `<b>` tags within the paragraph under this sub-heading.
- **Sub-Heading 5:** Use the `<h6>` tag.  
  Include an image using the `<img>` tag and underline text using the `<u>` tag within the paragraph under this sub-heading.

### Additional Instructions

- **Container Divs:** Wrap all headings and their associated paragraphs in `<div>` tags for proper separation and styling.
- **Inline Elements:** Add a `<br>` tag after each inline element (e.g., `<i>`, `<strong>`, `<em>`, `<b>`, `<u>`) for proper formatting.
- **Block Elements:** Do not add any extra tags after block elements (e.g., headings, paragraphs).
